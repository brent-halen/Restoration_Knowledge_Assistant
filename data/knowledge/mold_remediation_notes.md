# Mold Remediation Notes

## When this is urgent

Small, isolated growth without active water intrusion is usually not an emergency, but active leaks, widespread growth, strong odor, or occupant health concerns increase urgency.

## First actions

1. Address the moisture source first.
2. Limit disturbance if there is visible growth.
3. Use appropriate PPE for cleanup work.
4. Separate clean and contaminated materials during handling.

## Safety

Do not overstate certainty from remote descriptions alone. If contamination appears widespread or occupants report symptoms, recommend professional inspection and remediation planning.

## Demo note

This repo uses summary guidance for a portfolio project. It is not a substitute for licensed, site-specific assessment.
